Inventaire des Alertes - DataPass

Cette page affiche tous les types d'alertes possibles dans les vues demande, habilitation et résumé.

Correspondance avec ALERTS_INVENTORY.md

Chaque alerte ci-dessous est documentée avec :

  • Pages: Les vues où l'alerte peut apparaître (Demande, Habilitation, Résumé avant soumission)
  • Fichier: Le fichier source de la vue/partial
  • Condition: Les conditions Ruby qui déclenchent l'affichage de l'alerte
  • Clé i18n: La clé de traduction utilisée
  • Note: Informations complémentaires si nécessaire

Cette documentation correspond exactement à la structure détaillée dans ALERTS_INVENTORY.md.

Clé i18n: authorization_requests.show.dirty_from_v1.title
Note: Affiche une liste d'erreurs si @authorization_request.dirty_related_errors.any?

Cette demande d'habilitation ou habilitation est importée de la v1 de DataPass

Celle-ci présente des données incomplètes, manquantes ou corrompues. Par conséquent, certaines actions ne sont pas disponibles, telles que la réouverture ou le transfert. Si vous voulez effectuer des actions sur cette demande ou habilitation, merci de contacter le support à l'adresse datapass@api.gouv.fr

Les erreurs suivantes ont été rencontrées :

  • Le numéro SIRET de l'organisation n'a pas pu être récupéré lors de l'import
  • La date de création de la demande est au format invalide
  • Certains contacts techniques n'ont pas pu être importés
